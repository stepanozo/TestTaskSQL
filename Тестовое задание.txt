В решении есть файл DataBase, в котором необходимо указать базу данных.
Задание выполнено на C#, MS SQL Server.


ТЕСТОВОЕ ЗАДАНИЕ:

База данных склада – показывает наличие, приход и расход товара.

БД: таблицы товар, приход, расход.
Товар: наименование. Приход: дата, товар, количество.
Расход: дата, товар, количество.

Программа: отображение наличия товара, прихода и расхода. Редактирование таблицы товаров; запретить дублирование наименований с помощью логики программы.
Приход и расход: добавление, изменение и удаление прихода и расхода. Сделать возможность ограничения вывода прихода/расхода по периоду дат.

Cделать программу на Delphi или C#, СУБД MS SQL Server или MS Access. Следуйте стандартным практикам создания БД, реализуйте один из типичных вариантов интерфейса по своему выбору.
